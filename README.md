# BoxActions

BoxActions transforms incoming messages into ready-to-validate actions.

## V1
SMS inbox with AI-generated reply suggestions.

Flow:
1. A message is received
2. AI suggests a reply
3. User can approve, edit, or cancel
4. If approved, the SMS is sent

## Long-term vision
A message-first platform that can connect SMS, email, WhatsApp, calendar, invoices, follow-ups, and more.
